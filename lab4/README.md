# Лабараторная праца 4: Растравыя алгарытмы

Праца выканана ў выглядзе інтэрактыўнай графічнай прылады для візуалізацыі і аналізу растравых алгарытмаў, напісанай на Python з выкарыстаннем бібліятэкі PyQt5 для стварэння інтэрфейсу.

Прылада прадстаўляе сабой інтэрактыўны плотэр для пабудовы і аналізу адрэзкаў і акружнасцей з выкарыстаннем розных растравых алгарытмаў. Яна дазваляе параўноўваць эфектыўнасць алгарытмаў, назіраць працэс пабудовы і аналізаваць матэматычныя разлікі.

## Рэалізаваныя алгарытмы

### Асноўныя алгарытмы пабудовы адрэзка:
- **Пакрокавы алгарытм** - выкарыстоўвае ўраўненне прамой з акругленнем каардынат
- **Алгарытм ЦДА (Digital Differential Analyzer)** - лічбавы дыферэнцыяльны аналізатар
- **Алгарытм Брезенхема** - з выкарыстаннем цэлалікавай арыфметыкі
- **Алгарытм Кастла-Пітвея** - аптымізаваная версія алгарытму Брезенхема

### Алгарытмы пабудовы акружнасці:
- **Алгарытм Брезенхема для акружнасці** - з выкарыстаннем восьмай часткі акружнасці

### Алгарытмы зглажвання (анты-аліасінг):
- **Алгарытм Ву** - з выкарыстаннем інтэнсіўнасці пікселяў
- **Згладжаны пакрокавы алгарытм** - з разлікам доляў пікселяў
- **Гаусава зглажванне** - з выкарыстаннем Гаусавага ядра
- **Супер-семплінг** - з павышанай дыскрэтызацыяй

## Матэматычныя асновы

### Ураўненне прамой:
```
y = k × x + b, дзе k = (y₂ - y₁)/(x₂ - x₁), b = y₁ - k × x₁
```

### Алгарытм Брезенхема (для |k| < 1):
```
Пачатковая памылка: error = 2 × dy - dx
Абнаўленне памылкі: error += 2 × dy
Карэкцыя каардынаты: калі error ≥ 0, то y += y_step, error -= 2 × dx
```

### Алгарытм Брезенхема для акружнасці:
```
Пачатковая памылка: d = 3 - 2 × r
Абнаўленне: калі d > 0, то y -= 1, d += 4 × (x - y) + 10
           інакш d += 4 × x + 6
Заўсёды: x += 1
```

### Алгарытм Ву (зглажванне):
```
Інтэнсіўнасць = доля аддалення ад ідэальнай прамой
Для кожнага пікселя: 2 пункты з рознымі інтэнсіўнасцямі
```

## Тэхнічныя асаблівасці

### Выкарыстаныя бібліятэкі:
- **PyQt5** - стварэнне графічнага інтэрфейсу
- **Math** - матэматычныя вылічэнні
- **Time** - вымярэнне часу выканання

### Асаблівасці рэалізацыі:
- Усе алгарытмы рэалізаваны ўручную з выкарыстаннем кастамных формул
- Магчымасць параўнання часу выканання алгарытмаў
- Інтэрактыўны інтэрфейс з магчымасцю маштабавання і перамяшчэння
- Візуалізацыя сеткі з каардынатнымі восямі
- Падрабязныя матэматычныя разлікі для кожнага алгарытму

### Дадатковыя функцыі:
- **Вымярэнне часу выканання** - параўнанне эфектыўнасці алгарытмаў
- **Падрабязная статыстыка** - колькасць пікселяў, разлікі параметраў
- **Магчымасць захавання** - захаванне вынікаў і разлікаў
- **Інтэрактыўнае кіраванне** - мышкай і колечкам

## Запуск праграмы

Праграма падтрымлівае запуск як лакальна, так і з выкарыстаннем Docker кантэйнера. Усе неабходныя файлы для запуску ўключаны ў праект.

### Лакальны запуск:
```
python main.py
```

### Запуск з Docker:
```
docker run -it --rm -e DISPLAY=$DISPLAY -v /tmp/.X11-unix:/tmp/.X11-unix:rw jlcvuryj/raster-algorithms
```

Праграма даступная на DockerHub: `jlcvuryj/raster-algorithms`
